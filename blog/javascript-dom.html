JavaScript and its relationship to HTML and CSS

What is the DOM?

A programming interface where we can change things inside the document such as structure, style and content represented as nodes and objects so we can interact with them. The DOM is not a JS language but instead a web API to build websites. While the DOM is usually used with JS it can be built for any language. An example of this interaction would be when submitting a form when you click the submit button the dom takes in the data and gives back a result usually something like signing you in or completing an order.

If we were to do number 3 first it messes up our execution and we get a different result then the one we want. Loops are how we can repeat a process multiple times for example on a music playlist the music loop ends when we reach the end of the playlist and there isn't any more songs to play.
